# smart-resume-parser
Smart Resume Parser using Python, Streamlit and PDF Processing
# Smart Resume Parser

## Objective

Automatically extract information from PDF resumes.

## Features

- Resume Upload
- Email Extraction
- Phone Extraction
- Skill Detection
- Resume Scoring
- CSV Report Generation

## Technologies Used

- Python
- Streamlit
- Pandas
- PyMuPDF
- Regex

## Run

pip install -r requirements.txt

streamlit run app.py
